# 用于描绘训练曲线图  

## 一、 目标 
1. 根据训练重量、动作、组数描绘肌肉维度曲线  
...  
  
## 二、 需求分析  
* 可变量：  
> 动作名称（包含主要训练部位，相关肌肉的维度）。  
> 训练重量 。  
> 组数。  
  
* 关键量  
> rms: 最大重复次数  
  
* 输入量  
> 动作名称  
> 重量  
> 组数  
  
* 输出量  
> 相关肌肉名称（按发力优先级排序）  
> 肌肉维度
